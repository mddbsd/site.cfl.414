# Landing page y galería de cursos para el CFL 414

Esta es la nueva página del Centro de Formación Laboral 414, es una landin page con una galería de cursos y formularios de inscripción. Los cursos van a ser objetos en un archivo JSON y la información del formulario se carga en una planilla de google.

Es provisional hasta que esté preparado el backend con un panel de usuarios y administración
